nix #

vtamer/nix is Linux/BSD related code

/etc/passwd

| Code | Field | Description |--------|----------|------------------------------------------------------ | 0 | Username | The unique name used for logging in (1-32 characters). | 1 | Password | An 'x' indicates the password is stored in the /etc/shadow file. | 2 | UID User | Identifier, a unique number assigned to each user. | 3 | GID Group | Identifier, indicating the user's primary group. | 4 | GECOS | General information about the user, often the full name. | 5 | Home | Directory The path to the user's home directory. | 6 | Shell | The default shell that starts upon login.

/etc/group

| Code | Field | Description ||--------|------------|-----------------| 0 | group_name | The name of the group, which must be unique. || 1 | password | This field can contain an encrypted password. If it shows an 'x', shadow passwords are used. || 2 | GID | The Group ID, a unique numerical identifier for the group. || 3 | user_list | A comma-separated list of usernames that are members of the group. |
