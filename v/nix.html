nix #

tamer/nix

tamer/nix is Linux specific code, which some of it may run on bsd. It avoids distro and systemd specific code to be universaly Linux compadable.

alpha

This code is in alpha, do not use in production!

processes()

Aggregates TCP and UDP listeners, then iterates to replace "inode=XXXX" with the process name if found. Returns an array of strings like "tcp 0.0.0.0:22 sshd", or leaves inode if not found; also leaves inode if execution isn't done by root. (check the cmd folder for more info on the command, and the docs folder for the module manual)

cmd/timezone-dst/timezone-dst.v

Prints out timezone without considering dst; and N for North, S for South of equator. example output +03:00N

user.v

user dives in to /etc/passwd and /etc/group to get info. It has many helper functions such as:- fn user_has_home(username string) bool checks if the username has a home directory according to /etc/passwd

  • fn user_in_group(username string, group_name string) bool checks if the user has given group privliages
  • check the docs in the doc folder ...

v install --git https://codeberg.org/tamer/nix
