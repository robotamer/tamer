markdown #

markdown

Markdown module for V. For now, it uses the md4c library for parsing markdown files to HTML and soon it will be rewritten from scratch to pure V.

Usage

module main
import markdown

fn main() {
    text := '# Markdown Rocks!'
    output := markdown.to_html(text)
    println(output) // 
Markdown Rocks!

}

Installation

Install and use markdown module via VPM:

v install markdown

Or via vpkg:

vpkg get https://github.com/vlang/markdown --global

Or through Git:

git clone https://github.com/vlang/markdown.git ~/.vmodules/markdown

interface HtmlTransformer #

interface HtmlTransformer {
	transform_attribute(parent ParentType, name string, value string) string
	transform_content(parent ParentType, text string) string
mut:
	config_set(key string, value string)
}

HtmlTransformer transforms converted HTML elements (limited to attribute and content for now) from markdown before incorporated into the final output.

When implementing an HtmlTransformer, escaping the content is up to you. However, it is best to wrap the values you don't need with markdown.default_html_transformer.transform_{attribute|content}
