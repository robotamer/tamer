gittool #

This is called gittool, but it turned out to be a bid more then just a git tool.

  1. A module (gittool) and a command line tool (commit); to git add, git tag, git commit, and git push.Including auto incrementing tags with semantic versioning.

  2. A convention based build system largly for V language, although there is some flexibility build in so in can be used for other languages as well.

Installation

v install https://codeberg.org/tamer/gittool.git
./make.vsh help

Usage of commit as standalone cli command

$ commit help
Usage: commit [flags] [commands]

Git add, tag, commit & push

Flags:
  -m  -message        
  -d  -dryrun         
  -i  -minor          increment minor
  -j  -major          increment major
      -help           Prints help information.
      -man            Prints the auto-generated manpage.

Commands:
  info                Print information
  status              Print git status
  help                Prints help information.
  man                 Prints the auto-generated manpage.

Usage as module

  • See doc folder for module documentation
  • See make.vsh as example.
  • Use make_temlate.vsh as template.

The convention is simple.

Put all your commands in to ./cmd/folder/files.v

running ./make.vsh build

Each folder becomes a binary, ./bim/foldername

running ./make.vsh release

Each folder becomes a binary, ./bim/foldername-version-os-statis

$ ./make.vsh help
: help
Tasks:
- help: Print help (default)
- fmt: Run fmt on all v files
- build: Build for dev
- release: Build universal static binaries
- uninstall: Remove symlink from bin dir
- doc: create markdown docs in ./doc
- vsh: re-build make.vsh
- info: View git and vmod info
- commit: git commit (Usage: ./make.vsh commit [-p] your message here
      default: patch; release: minor & push)
import build
import v.vmod
import gittool

const manifest = vmod.from_file('v.mod') or { panic('Failed to decode v.mod: ${err}') }

// Make the build context
mut context := build.context()
// help executes --task and is default if no args provided
gittool.context_help(mut context)
// context V is seperated so the build system can be used with other coding languages
gittool.context_v(mut context) 
gittool.context_git(mut context, manifest)
