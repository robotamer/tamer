upgrade #

upgrade

A lightweight V module to automatically rebuild and restart your application when the version in v.mod changes. Useful for development workflows, and self-updating applications.

Features

  • Native V: No external dependencies beyond V standard library.
  • Compile-time embedding: Version is baked into the binary at build time.
  • Automatic detection: Compares embedded version with on-disk v.mod on startup.
  • In-place rebuilds: Rebuilds the executable and restarts seamlessly.
  • Remote sync: Pull latest changes and rebuild with from_remote() repo.

Documentation

  • browser friendly docs

Installation

v install https://codeberg.org/tamer/upgrade.git

Usage

import upgrade
import v.vmod

// Embed at compile time, manifest and git dir location to track the version the binary was compiled with

// --- MANIFEST backed in to executable ---
const manifest_data = $embed_file('v.mod')
const manifest = vmod.decode(manifest_data.to_string()) or {
    panic('Failed to decode embedded v.mod: ${err}')
}
const dir_git = os.getwd()

fn main() {
    upgrade_manager := upgrade.run(dir_git, manifest)!

    // Your application logic here
    println('Running version ${upgrade_manager.version}')
}

Check for updates manually

upgrade_manager.check()!

Pull remote changes and rebuild

upgrade_manager.from_remote()!
